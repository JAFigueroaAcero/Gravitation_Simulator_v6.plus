Para ejecutar el programa:
-Se debe tener al menos python 3.
-Instalar las librerías de: matplotlib, numpy, pandas, bokeh, tkinter, os y sys.

NOTA:
-Para ejecutar la información desde cero ejecutar el script interfaz.py
-Para cargar los datos anteriores ejecutar launch.py
-La modificación de los nombres podría afectar la funcionalidad del simulador
-Se recomienda usar valores conocidos para evitar choques entre sistemas
